# School Management System

A Spring Boot based web application to manage Students and Teachers with CRUD operations.


## 1. Project Overview
This application allows users to manage Students and Teachers.  
It supports adding, viewing, editing, and deleting records using a clean MVC architecture.


## 2. Technologies Used
- Java
- Spring Boot
- Thymeleaf
- MySQL
- HTML, CSS


## 3. Project Structure
The project follows a standard Spring Boot structure:
- controller – handles HTTP requests
- service – business logic
- repository – database access
- model – entity classes
- templates – HTML files
- static – CSS files


## 4. Database Setup
- MySQL is used as the database.
- SQL schema is provided in database/schema.sql.
  

## 5. How to Run the Project
   Run `SchoolApplication.java`
   Open `http://localhost:8080` in browser
